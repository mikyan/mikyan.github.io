git仓库迁移后不显示部分contributor解决办法

2020-12-14 2 min read # git

问题的产生

学校的小学期项目托管在自建的gitlab上，小学期结束想把仓库迁移到github，按照这篇文章

从一个git仓库迁移到另外一个git仓库

进行了迁移，但是迁移后，contributors里面居然没我，网上查了一下，没有查到解决方案。

解决办法

检查之后发现其实提交记录里是有我的，但是github没有将我的提交记录映射到我的github账号。

想到我之前修改了github用的邮箱，并把老邮箱账号删除了，然后注意到我的的ssh-key是用之前的qq邮箱账号生成的，里面包含我邮箱的字段，猜想github是不是靠这个来识别用户的，于是在github里把原来的邮箱添加回去，再重新执行迁移过程，果然解决了问题。

不知道具体原因是不是和我猜想的一样，可能需要多了解git的原理才能弄清，以后如果更深入的学习了git，解决了疑惑，可能会过来详细写写原因。
